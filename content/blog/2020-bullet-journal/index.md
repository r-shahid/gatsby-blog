Here, I want to address the way a bullet journal is ‚Äúsupposed to‚Äù be used. A bullet journal usually has weekly pages or daily pages. If you have a task that you keep pushing to the next day, you should realize that you‚Äôre either not being realistic about how much you can get done in one day or you should re-evaluate how worthy that task is at all. If you need to keep rescheduling it, and you don‚Äôt have any consequences of not completing the task yet, then ask yourself if you really need to do it or if it‚Äôs something you can delegate.

##### Weekly Layouts

There are COUNTLESS examples of different layouts online, but here are four that I think are very easy to setup and look cute without compromising on function.

The first week is a two-page spread. I left space open to write my daily to do lists and put some pictures in the blank spaces.
